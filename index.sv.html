PiEdge EduKit — Självständig mikro-lektion (~30 min)

Syfte: Träna en liten bildklassificerare, exportera till ONNX, benchmarka CPU-latens och jämför FP32 vs INT8. Avsluta med automatisk verifiering som skriver JSON-kvitto.

English version

Tidsbudget

Cirka ~30 minuter aktiv tid, med snabba standardvärden (1 epoch, små batchar).

Förutsättningar

  • OS: Ubuntu 22.04 / macOS (Intel/Apple Silicon) / Windows 11
  • Python 3.12 installerad
  • Terminal. Inga admin-installationer behövs utöver lokal venv.

Steg 1 — Skapa och aktivera .venv

bash scripts/setup_venv.sh
# Aktivera (Linux/macOS):
source .venv/bin/activate
# Aktivera (Windows PowerShell):
.venv\Scripts\Activate.ps1
TipsOm pip saknas, kör python -m ensurepip --upgrade innan installation.
Alternativ med uv: uv venv && source .venv/bin/activate && uv pip install -r requirements.txt

Steg 2 — Kör lektionen (självständig)

bash run_lesson.sh

Detta kör: träning (FakeData), export, benchmark, kvantisering, utvärdering och till sist auto-verifiering.

Steg 3 — Titta på resultaten

  • reports/latency_summary.txt
  • reports/quantization_comparison.csv
  • reports/confusion_matrix.png, reports/eval_summary.txt
  • progress/receipt.json (auto-kvitto)

Automatisk verifiering

Kör när som helst:

python verify.py

Uppdaterar progress/receipt.json och returnerar PASS/FAIL med tydliga skäl.

Vanliga fallgropar

  • Apple Silicon: onnxruntime-silicon installeras automatiskt via miljömarkörer.
  • Windows PowerShell: Kör Set-ExecutionPolicy -Scope CurrentUser RemoteSigned för att möjliggöra venv-skript.
  • Windows venv-problem: Om bash scripts/setup_venv.sh ger permissions-fel, testa py -3.12 -m pip install --user -r requirements.txt och py -3.12 -m pip install --user -e . i stället.
  • Python 3.12: Kontrollera installationen med python --version.

Packa för inlämning

Förslag på zip-namn: Lesson_PiEdge_YYYYMMDD.zip. Inkludera hela projektroten.

Använd: bash scripts/lesson_pack.sh för automatisk packning.
